A preparatory checklist for attendees of a Quarto + GitHub Pages workshop.
It walks through installing Git on Windows, Mac, and Linux, creating a GitHub
account with a personal access token, configuring Git locally, and wiring
everything into RStudio.

The point of writing this out: the first ninety minutes of any workshop
disappears into "my install doesn't work" if attendees haven't done the setup
beforehand. This post moves that work to the day before.
