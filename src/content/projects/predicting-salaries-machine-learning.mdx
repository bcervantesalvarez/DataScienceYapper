A team ML project classifying whether an individual earns more than $50K from
the OpenML census dataset. The pipeline runs preprocessing, PCA-based feature
selection, and Random Forest tuning to a Kappa of 0.9543.

Mostly useful as a template for how to structure a small applied-ML project: a
clean train/validation split, an explicit feature-selection step, and a single
ranked-model output that you can defend to a reviewer.
