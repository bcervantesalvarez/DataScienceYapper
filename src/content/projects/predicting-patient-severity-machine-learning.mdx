<Overview>
  A classifier for rare-disease severity from symptom flags. The non-obvious requirement: the output had to be a handful of human-readable rules clinicians could sanity-check, not a black-box probability score.
</Overview>

## Notes from the archive

Built in Python with scikit-learn. The Random Forest itself was straightforward; the real work was distilling it down to a small rule set that approximated physician judgement well enough to be useful as a triage aid. Evaluated with confusion matrix, ROC AUC, precision-recall, and Cohen kappa to confirm the simplified rules did not trade away too much performance.
